## Level 13 → Level 14

### Problema

The password for the next level is stored in
**/etc/bandit_pass/bandit14 and can only be read by user
bandit14**. For this level, you don’t get the next password, but you
get a private SSH key that can be used to log into the next level.
**Note:** **localhost** is a hostname that refers to the machine
you are working on

### Notas

En este laboratorio la clave para estaba en otro usuario, pero habia una llave ssh que se podia utilizar para entra al host donde se encotraba la llave, para esto se ocupo ssh -i que nos ayuda a poner la ubicacion de la llave antes del host: **ssh -i ruta_a_la_clave_privada usuario@hostname**

### Comandos

| Comandos | Funcion |
| --- | --- |
| ssh -i | -i: Indica que seguirá la ruta al archivo de clave privada que deseas usar. |
